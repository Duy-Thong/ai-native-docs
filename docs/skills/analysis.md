# Skills Phân tích (Analysis)

Workspace này không có skills chuyên biệt cho phase phân tích. Công việc phân tích yêu cầu được thực hiện trực tiếp qua chat với Copilot, hoặc thông qua các skills trong nhóm [Thiết kế](./design) (document workspace).

> Nếu cần phân tích yêu cầu từ file scope/ticket, hãy dùng `detail-design-api-creator` hoặc `detail-design-screen-creator` — các skills này đều hỏi xác nhận khi còn điểm chưa rõ trước khi finalize thiết kế.
